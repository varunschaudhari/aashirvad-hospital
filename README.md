# Aashirvad Hospital Website

A simple, standalone HTML website for Aashirvad Hospital.

## Features

- Responsive design
- Mobile-friendly navigation
- Smooth scrolling
- Contact form
- No build tools required
- No npm dependencies

## How to Use

Simply open `index.html` in your web browser. That's it!

No installation, no build process, no npm required.

## File Structure

```
.
├── index.html          # Main HTML file
├── src/
│   ├── main.js        # JavaScript for interactivity
│   └── assets/        # Images
└── README.md          # This file
```

## Technologies Used

- HTML5
- CSS3 (with Tailwind CDN)
- Vanilla JavaScript
- Google Fonts (Nunito & Inter)

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
